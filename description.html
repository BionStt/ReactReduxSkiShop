Introduction

This React Redux Single Page Application builds 

  1. A product navigation system that customers can use to browse skis by  category, price, gender, ideal- for, and page
  2. A shopping cart where customers can add and remove skis
  3. A checkout where customers can fill in their shipping details and place their orders
  4. An order list where customers can review their orders.

In this sample, we use

  • React and its compatible packages as front-end platform
  • ASP.NET Core as server-side platform

On the backend, we use

  • EF Core Code First to build a database on MS SQL Server Express and seed its data
  • ASP.NET Core Web API to reach clients

On the client-side, we use

  • Webpack to bundle client-side resources (HMR for development)
  • React components to compose the UI
  • Redux to manage states (redux thunk for async calls)
  • React Router 4 for routing (qs for query strings)
  • Babel to compile JSX and make use of latest ES6 features
  • Bootstrap4 for styling and layout (font-awesome for glyphicons)

Notice: IIS settings in the web.config for react routing system cannot work with Hot Module Replacement. Change its name from web2.config to web.config when production needed.

Prerequirements:

  • Visual Studio 2017
  • ASP.Net Core 1.1.0 and up
  • EF Core
  • Node.js
  • ReactJS and other additional modules

import React from 'react';
import PropTypes from 'prop-types';
import { Link } from 'react-router-dom';

import IfClearancePrice from './IfClearancePrice';

const Skis = ({ skis, onClick }) => (
    <div className="col-4 mt-3">
        <Link to={`/styleDetail/${skis.categoryName}/${skis.styleNo}`}
            className="pb-3 mb-3" style={{textDecoration: 'none'}}>
            <img src={skis.imageSmall} onClick={onClick}/>
        </Link>
        <Link to={`/styleDetail/${skis.category}/${skis.styleNo}`}
            style={{fontSize: '0.70rem'}}>
            <p className="mb-0 text-muted">{skis.brandName.toUpperCase()}</p>
            <strong style={{ color: 'black' }}>{skis.styleName} - {skis.gender}</strong>
            <IfClearancePrice current={skis.priceCurrent} regular={skis.priceRegular} />
        </Link>
    </div>
);

Skis.PropTypes = {
    skis: PropTypes.shape({
        styleNo: PropTypes.string.isRequired,
        styleName: PropTypes.string.isRequired,
        brandName: PropTypes.string.isRequired,
        gender: PropTypes.string.isRequired,
        categoryName: PropTypes.string.isRequired,
        imageSmall: PropTypes.string.isRequired,
        priceCurrent: PropTypes.number.isRequired,
        priceRegular: PropTypes.number.isRequired
    }).isRequired, 
    onClick: PropTypes.func.isRequired
}

export default Skis;
